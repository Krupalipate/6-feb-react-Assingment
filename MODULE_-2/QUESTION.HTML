Writing into an alert box, using window.alert(). Writing into the browser console, using console.log(). Here's an console.log("Hello world") [2] document.write(): This method is used to write HTML content directly to the document. Click Event: To trigger a function when a button is clicked, you can use the addEventListener() method to add a click event listener to the button element. const button = document.querySelector('#myButton'); button.addEventListener('click', () => { // code to be executed when the button is clicked }); 2. Mouseover Event: To trigger a function when the mouse pointer is moved over an element, you can use the mouseover event. const element = document.querySelector('#myElement'); element.addEventListener('mouseover', () => { // code to be executed when the mouse is moved over the element }); 3.
